## 📖 About

Project Genesis is a structured 90-day learning journey documenting my transition from a beginner in Web3 to a blockchain developer.

This repository contains my daily notes, exercises, projects, reflections, and technical documentation as I learn blockchain technology, smart contract development, decentralized applications (dApps), and the broader Web3 ecosystem.

The goal is not only to learn Web3 but also to build a public portfolio that demonstrates consistent growth, practical skills, and professional documentation.

---

## 🎯 Objectives

- Learn Blockchain Fundamentals
- Understand Ethereum Architecture
- Master Solidity
- Build Smart Contracts
- Develop Full-Stack dApps
- Contribute to Open Source
- Build a Professional Portfolio
- Secure My First Web3 Opportunity
